<h1>Orientação a Objetos - PHP</h1>

e manutenção de uma Conta Bancária.</p>

<p>Os estudos envolvem:</p>
<ul>

  <li>Encapsulamento</li>
  <li>Herança</li>
</ul>

<p>Também estudei:</p>
<ul>

  <li>Arquvivos separados para cada Classe</li>
  <li>Require e Use</li>
  <li>Namespaces</li>
</ul>
